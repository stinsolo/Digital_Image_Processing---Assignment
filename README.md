A comprehensive collection of Digital Image Processing techniques implemented using Python.
Features include Image Enhancement, Histogram Processing, Spatial Filtering, Image Sharpening,
Frequency Domain Filtering, and Manual DFT Implementation.

**Installation** :
git clone https://github.com/stinsolo/Digital_Image_Processing---Assignment.git
pip install opencv-python scikit-image matplotlib numpy scipy ipywidgets

**Technologies** :
Python, OpenCV, NumPy, Matplotlib, SciPy, Scikit-Image, IPyWidgets
